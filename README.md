# vpn-gptray
A GTK3 system tray application for the GlobalProtect VPN client

![Tray dropdown](/doc/screenshot2.png)

## Installation
Download and unpack the zip file.

Run `install.sh` to install. Afterward, a **vpn_gptray** icon should appear in your Gnome App Menu

![App icon](/doc/screenshot1.png)

Alternatively, start the app by running `vpn_gptray` (already in PATH if the installation was successful)

## Customization
`~/.config/vpn-gptray/config.txt`
